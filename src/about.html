Likes Feature Functionality

Function Parameters Description
export async function getLikes() N/A Fetches a table of all "likes" and returns the "likesList".
export async function getLikesByPostID(postID) postID Fetches a table of "likes" based on a "postID" of a specific post and returns the "likesList".
export async function addLike(userId, postId) userId, postId Fetches a table of "likes" and uses method "POST" in order to add a like based on the logged-in user (userId) and the id of the post (postId).
export async function removeLike(userId, postId) userId, postId Fetches a specific "like" from the table of "likes" based on the logged-in user (userId) and the id of the post (postId) in order to determine the entry of the "like" on the table then remove it based on the entry number.
